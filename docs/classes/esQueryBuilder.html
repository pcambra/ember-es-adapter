addSort

(
  • sort
)

Defined in addon/utils/es-query-builder.js:125

Adds a sort param. Takes a string or object. These are stackable so multiple sorts can be applied.

Example use:
let es, myQuery;
es = new esQuery();
es.addSort('title');
es.addSort({'date' : { 'order': 'asc'}});
es.getQuery(); 

Example Sorts
simple sort
"user"

less simple sort
{ "name": "asc" }

complexish simple sort
{ "post_date": { "order": "asc"} }

Parameters:

  • sort Object

    Sort object/string to be added.
